# TicketRush

## Swagger

Sau khi chay backend, mo Swagger UI tai:

`http://localhost:8082/swagger-ui/index.html`

OpenAPI JSON:

`http://localhost:8082/v3/api-docs`

Neu endpoint can dang nhap:

1. Dang nhap de lay bearer token/JWT.
2. Mo nut `Authorize` tren Swagger UI.
3. Nhap theo format: `Bearer <your-token>`.
