
  # Thiết kế giao diện web

  This is a code bundle for Thiết kế giao diện web. The original project is available at https://www.figma.com/design/6geNInuWhi23sCkWfhlLOJ/Thi%E1%BA%BFt-k%E1%BA%BF-giao-di%E1%BB%87n-web.

  ## Running the code

Run `npm i` to install the dependencies.

Run `npm run dev` to start the development server.

If you want the QR screen to include a clickable online verification link, set `VITE_PUBLIC_APP_URL` to a reachable frontend URL before running the app. The QR itself still shows the order and transaction summary even without that variable.
